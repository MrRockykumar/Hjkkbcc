Install on Your Android Device

Download the compiled Android APK to install RTV+ on any phone, tablet, or TV.

1

Get the APK

Click below to download the latest compiled Android debug APK file. If you are inside the AI Studio Web editor, click the Settings icon in the top-right to download the built APK directly to your machine.

Local app-debug.apk
2

Enable Unknown Sources

Android devices protect against untrusted apps. To install the APK, navigate to Settings > Security > Install unknown apps and grant permissions to your browser or file manager.

3

Launch & Watch

Tap the downloaded APK in your device notifications or Downloads folder to run the Android Package installer. Tap "Install anyway" if Play Protect warns about an unsigned debug package. Once complete, launch the RTV+ App from your launcher!
